# drop/ — 신곡 등록용 드롭 폴더

**한 곡당 한 하위폴더** 를 만들고, 그 폴더 안에 **MP3 1개 + Word 가사(.docx) 1개**를 넣으세요. 그리고 Claude Code에게 "등록해줘"라고 하면 됩니다.

## 폴더 구조

```
drop/
├─ 서두르지 않아/
│  ├─ 서두르지 않아(OVERHEAT).mp3
│  └─ 서두르지 않아(OVERHEAT)_가사.docx
├─ 다음 곡/
│  ├─ 다음 곡 (Next).mp3
│  └─ 다음 곡 (Next)_가사.docx
└─ 세번째 곡/
   ├─ ...
   └─ ...
```

- 폴더명은 한글이든 영문이든 자유. 본인이 알아보기 쉬운 이름이면 OK.
- 한 폴더 안에는 **MP3 1개 + .docx 1개** 만. (다른 파일이 있어도 등록 후 함께 processed/ 로 이동됨)
- 폴더가 여러 개면 Claude가 **발견 순서대로 순차 등록** 합니다.

## 파일 명명 규칙 (폴더 내부)

권장 형식:
```
원곡제목 (보조제목).mp3
원곡제목 (보조제목)_가사.docx
```

- 한국어 곡 → 한글 제목 (영문 부제)
- 외국어 곡 → 원곡 제목 (한국어 번역)
- 괄호 안이 비어 있어도 OK — Claude가 적절한 번역 제목을 제안

## 가사 파일 형식 (.docx)

**원곡 언어**로 작성:
- 빈 단락(엔터 2번)으로 절 구분
- 각 절의 첫 줄에 라벨(`1절`, `후렴` 등) — 또는 라벨 없이 빈 단락만으로 구분도 OK
- 글꼴/색상 등 스타일은 무시되고 텍스트만 추출
- Word에서 **저장하고 닫은 뒤** 폴더에 넣어주세요 (`~$파일.docx` 잠금 파일 충돌 방지)
- 반복되는 후렴은 한 번만 적어도 OK — 어차피 앱은 유니크 섹션만 표시

## 번역

원곡 가사가 어느 언어로 작성되어 있든 (한국어/영어/스페인어/일본어), Claude가 **나머지 3개 언어로 자동 번역** 해서 총 4개 언어 버전을 앱에 반영합니다. 별도 번역 파일을 준비할 필요 없음.

## 처리 후

등록 성공 시 곡 폴더 전체가 `drop/processed/{트랙번호}/` 로 이동됩니다. 이 폴더는 git에 커밋되지 않아요.
