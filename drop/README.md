# drop/ — 신곡 등록용 드롭 폴더

여기에 **MP3 1개 + 가사 Word 파일(.docx) 1개**를 넣고 Claude Code에게 "등록해줘"라고 말하면 됩니다.

## 파일 명명 규칙

권장 형식:
```
원곡제목 (보조제목).mp3
원곡제목 (보조제목).docx
```

예시:
```
봄날의 그림자 (Spring Shadow).mp3
봄날의 그림자 (Spring Shadow).docx
```

- 한국어 곡 → 한글 제목과 영문 제목
- 외국어 곡 → 원곡 제목과 한국어 번역 제목
- 괄호 안이 비어 있어도 OK — Claude가 적절히 번역 제목을 제안

## 가사 파일 형식

**MS Word 문서 (.docx)** 를 사용합니다. **원곡 언어**로 작성:

```
1절
첫 줄
둘째 줄

2절
첫 줄
둘째 줄

후렴
...

브리지
...

아웃트로
...
```

- Word 문서 안에서 **빈 단락(엔터 2번)** 으로 절을 구분
- 각 절의 **첫 줄**이 라벨로 사용됨 (예: `1절`, `후렴`, `Verse 1`, `Chorus`, `Verso 1`, `Estribillo` 등)
- 또는 라벨 없이 그냥 빈 단락으로만 구분해도 됨 — Claude가 자동으로 라벨 부여
- 표·각주·이미지는 무시됨. 순수 텍스트만 추출됨
- Word의 글꼴/색상 등 스타일은 무관 — 글자 내용만 가져옴

## 번역

원곡 가사가 어느 언어로 작성되어 있든 (한국어/영어/스페인어/일본어), Claude가 **나머지 3개 언어로 자동 번역**해서 총 4개 언어 버전을 만들어 앱에 반영합니다. 별도 번역 파일을 준비할 필요 없습니다.

## 처리 후

등록 성공 시 파일들은 `drop/processed/{트랙번호}/` 로 이동됩니다. 이 폴더는 git에 커밋되지 않습니다.
